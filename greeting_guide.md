# 打招呼语生成指南

## 核心原则

1. **分档定制**：重点岗按 JD 定制，普通岗用统一模板换公司/岗位名。
2. **去 AI 味**：禁止双引号 ""、箭头 ➡️ →、圆括号（）、直角引号「」、斜杠 /。短句口语、简洁真诚。
3. **首句成果钩子 ≤35 字**：不要"我对 XX 感兴趣"式寒暄开头，保证 BOSS 列表预览直接显示亮点。
4. **3-5 句，一屏读完**：太长 HR 没耐心看。
5. **末句 = 作品链接 + 轻召唤**：用 `{作品链接}` 占位，不放真实链接。

## 结构模板

```
[首句·成果钩子 ≤35字] 我做过[具体项目]，[量化成果]。
[第二句·JD对焦] 贵司这个岗做[JD提炼]，我用[相关技能]能直接上手。
[第三句·差异点] [与其他人不一样的地方，背景/证书/自驱等]。
[末句·作品+轻召唤] 作品说明在 {作品链接}，想跟您聊聊。
```

## 示例（占位替换后）

```
唐女士您好，我独立做过一个 RAG 文档问答系统，带溯源校验和 eval pipeline，已交付团队使用。贵司要用 AI 搭信息追踪系统和 Agent 开发自动化流程，我的落地经验应该能直接用上。作品说明在 {作品链接}，想跟您聊聊。
```

## 输出格式

输出为 Markdown 表，列名：

| 公司 | 岗位 | 城市 | 优先级 | HR 活跃状态 | 薪资 | 岗位链接 | 招呼语 |

- **优先级**填 重点岗/普通岗
- 每条招呼语带备注：用了哪个钩子 + 扣了 JD 哪个点
