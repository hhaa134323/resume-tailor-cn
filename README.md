# Resume Tailor CN

一个简历定制工具。你把目标岗位的招聘要求截图丢给 AI 助手，它就照着每个岗位把你的简历改好、排好版，导出成一页 A4 的 PDF。一次丢一批岗位，它一家一家帮你定制。整个过程你只用说人话，不用写任何代码。

它怎么运作，一句话：你的简历内容和排版是分开存的，内容放在文本文件里，排版样式已经调好固定住，一个脚本负责把内容套进模板生成 PDF。这些都由 AI 助手替你跑。

## 你需要准备什么

- 一台电脑
- 一个 AI 编程助手，Claude Code、Cursor、Cline 这类都行。
- 你现有的简历，用来当母版，第一次交给助手就行。

## 安装：直接让 AI 帮你装（推荐）

最省事的方式是让助手替你装。把这个仓库链接发给它，然后说一句类似的话：

> 帮我把 resume-tailor-cn 这个项目装成能用的简历技能。把整个仓库下载到本地放好，顺便把运行环境也配好。

它会把整个项目下载下来、放到合适的位置、装好需要的环境。

这里有个关键点：装的是整个项目，不是单独一个 SKILL.md。真正生成 PDF 的脚本、模板和样式都在项目里，只拷一个 SKILL.md 是跑不起来的。SKILL.md 的作用是告诉 AI 这套工具怎么用，它得配合完整的项目文件一起才有意义。

装完之后，问助手一句“现在能用 resume-tailor-cn 帮我改简历了吗”，确认它认得这个技能就行。

如果你想自己放，不让助手代劳，按你用的助手把整个项目放到对应位置：

- Claude Code：用 Claude Code 直接打开这个项目文件夹即可，它会自动读取里面的 SKILL.md 和 CLAUDE.md。也可以把整个 resume-tailor-cn 文件夹放进 ~/.claude/skills/ 当全局技能。
- Cursor、Cline、JetBrains 等：先把整个项目放进你的工作目录，再把 SKILL.md 的内容复制到该工具放规则的地方，Cursor 是 .cursor/rules/，JetBrains 是 .junie/guidelines/，让助手知道这个技能的用法。

## 怎么用：丢 JD 截图，拿定制简历

这是这个工具最日常的用法。

第一次使用，先把母版交给助手，告诉它“这是我的简历，先存成母版”，把简历文件或内容发给它。

之后每次投递：

1. 把目标岗位的招聘要求截图丢进对话框，可以一次丢一批，几家公司一起。
2. 说一句“按这些岗位分别帮我定制简历，每家各导出一份 PDF”。
3. 助手会一家一家来，读懂每个岗位的要求，调整你的个人定位、项目排序和技能重点，给每家公司生成一份单独的一页 A4 PDF。
4. 想更稳一点，再加一句“帮我跑一轮模拟 HR 评审”，或者直接输入 /hr-review。它会站在 HR 和用人经理的角度挑问题、改稿、重新导出。
5. 需要打招呼语时，说“帮我写这个岗位的打招呼语”即可。

## 你会拿到什么

成品是 PDF，存在项目的 cv 文件夹里。一页 A4，中文用楷体，英文和数字用 Times New Roman，排版已经固定调好。每家公司的内容互相独立，而且这些个人内容文件已经设置成不会被上传到 GitHub，不用担心信息泄露。

## 常见问题

- PDF 里中文变成方框或乱码：电脑缺楷体字体。跟助手说一句“中文显示成方框了，帮我换成系统里有的字体”就行。
- 内容一页放不下：跟助手说“太长了，帮我压到一页”，它会先收紧排版，还不够就帮你删掉最不相关的经历。
- 个人信息安全：示例文件里的姓名、学校、公司都是虚构的，你自己的内容文件不会被提交到 GitHub。
- 打招呼语不会自动发送：助手只负责写出来，发到招聘平台这一步要你自己复制粘贴。
